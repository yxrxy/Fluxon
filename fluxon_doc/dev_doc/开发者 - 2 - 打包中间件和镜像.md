# 开发者 - 2 - 打包中间件和镜像

本文只说明 `setup_and_pack/pack_release_ext.py` 和 `examples/fluxon_quick_start/build_image.py`。

## `ext_images/`

`pack_release_ext.py` 会导出这些对象到 `fluxon_release/ext_images/`：

- `etcd/etcd`
- `etcd/etcdctl`
- `etcd/start.sh`
- `greptime/greptime`
- `greptime/start.sh`
- `tikv/pd-server`
- `tikv/tikv-server`
- `tikv/start_pd.sh`
- `tikv/start_tikv.sh`
- `ext_images.sha256`

## 命令

```bash
python3 setup_and_pack/pack_release_ext.py --release-dir ./fluxon_release --with-tikv-runtime true
```

## Quick Start 镜像

`examples/fluxon_quick_start/build_image.py` 只有两种模式：

- `existing_release`
- `url_download`

### 现有 release

```bash
python3 examples/fluxon_quick_start/build_image.py --mode existing_release
```

### URL 下载

```bash
python3 examples/fluxon_quick_start/build_image.py \
  --mode url_download \
  --fluxon-wheel-url <url> \
  --fluxon-pyo3-wheel-url <url> \
  --pylib-src-url <url>
```
